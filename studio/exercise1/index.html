Exercise #1—Project page

Choose a past or current piece of work to share with the class. Gather various assets including: (a)images, videos or gifs that document the project (b)title and description text that describe the project (c)1 external link that's relevant to the inspiration of the project. Once you've gathered the assets, create a simple html project page.

Exercise Requirements

—use <h1> tag for titling text

—use <p> tag for description text

—use <i> or <b> tag at least once in your body text

—use <a> tag for an external link

—use <img> tag for images and gifs
